List of written methods for the Graph class:
    insertNote():
        In: A Node
        Out: Nothing
        Effect: Inserts a node into the Graph
    insertEdge():
        In: A Edge
        Out: Nothing
        Effect: Inserts a Edge into the graph between two nodes
List of written methods for the Node class:
    insertEdge():
        In: A Edge
        Out: Nothing
        Effect: Stores the Edge that is connected to the Node
